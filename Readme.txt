This is a simple Twitter application to stream the tweets to local file
It's build with the help of Twitter4J
It's using facade and factory design patterns for readability and better maintenance, also to avoid boilerplate code
User should provided the below values from Twitter Developer console after making a new app (link : https://developer.twitter.com/en/apps)
	 CONSUMER_KEY = "";
	 CONSUMER_SECRET = "";
	 ACCESS_TOKEN = "";
	 ACCESS_TOKEN_SECRET = "";
By default it will create the files in the execution directory.


Steps to run the application:
1) Execute SimpleTweets.bat file in the same directory where simpletweet.jar exists
2) Input five Tweet tags seperated by space
